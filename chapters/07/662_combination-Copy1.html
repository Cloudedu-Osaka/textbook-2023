組み合わせ: ピタゴラスの三数#

  • Pythagorean triple - Wikipedia

    • ピタゴラス数の求め方とその証明 | 高校数学の美しい物語

    • Plimpton 322 - Wikipedia

正の整数 \(p\), \(q\) \((p>q)\) に対して、

\[\begin{split} \begin{align} a & = 2 p q \\ b & = p^2 - q^2\\ d & = p^2 + q^2\\ \end{align} \end{split}\]

とおくと、\(a\), \(b\), \(d\) はピタゴラスの三数となり、\(a^2+b^2 = d^2\) を満たす。

解) 恒等式

\[ (x-y)^2 = (x+y)^2 - 4xy \]

に対して \(x=p^2\), \(y=q^2\) とおいて \(p\), \(q\) を消去する。

\[\begin{split} \begin{align} (p^2-q^2)^2 &= (p^2+q^2)^2 - 4p^2q^2 \\ b^2 &= d^2 - a^2 \end{align} \end{split}\]

Plimpton 322の最も左のカラム:

\[\left( \frac{d}{a} \right)^2 = \frac{p^2+q^2}{2pq} = \frac{1}{2}\left( \frac{p}{q} + \frac{q}{p} \right)\]

\(p > q\) のとき、\(\frac{p}{q}>1\) かつ \(\frac{q}{p} < 1\) が成り立つ

def decimal_to_babylonian(num):
    result = ''
    for value in sorted(babylonian_numerals.keys(), reverse=True):
        while num >= value:
            result += babylonian_numerals[value]
            num -= value
    return result

10進数を60進数に変換する関数を定義する#

%%ai_ask
pythonで10進数の実数を60進数に変換する。ただし、0から59までの数字のリストを出力する。

以下のようなコードで10進数の実数を60進数に変換し、0から59までの数字のリストを出力できます。

def decimal_to_sexagesimal(decimal):
    integer_part = int(decimal)
    fractional_part = decimal - integer_part
    sexagesimal_integer = []
    while integer_part > 0:
        remainder = integer_part % 60
        sexagesimal_integer.append(remainder)
        integer_part //= 60
    sexagesimal_integer.reverse()
    sexagesimal_fractional = []
    while fractional_part > 0 and len(sexagesimal_fractional) < 10:
        product = fractional_part * 60
        integer_part = int(product)
        sexagesimal_fractional.append(integer_part)
        fractional_part = product - integer_part
    return sexagesimal_integer, sexagesimal_fractional
