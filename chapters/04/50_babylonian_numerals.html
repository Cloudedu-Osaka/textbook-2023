def decimal_to_babylonian(num):
    result = ''
    for value in sorted(babylonian_numerals.keys(), reverse=True):
        while num >= value:
            result += babylonian_numerals[value]
            num -= value
    return result

60進数の位取り記法#

%%ai_ask
pythonで10進数の実数を位取り記法の60進数に変換する。戻り値は0から59までの整数のリストで返してください。

以下のようなコードで実現できます。

def decimal_to_sexagesimal(decimal):
    sexagesimal = []
    integer_part = int(decimal)
    decimal_part = decimal - integer_part
    
    while integer_part > 0:
        remainder = integer_part % 60
        sexagesimal.insert(0, remainder)
        integer_part //= 60
    
    for i in range(6):
        decimal_part *= 60
        sexagesimal.append(int(decimal_part))
        decimal_part -= int(decimal_part)
    
    return sexagesimal

この関数は、10進数の実数を引数として受け取り、60進数のリストを返します。整数部分と小数部分を分けて、整数部分を60進数に変換し、リストの先頭に挿入します。その後、小数部分を60進数に変換し、リストの末尾に追加します。最後に、60進数のリストを返します。

def decimal_to_sexagesimal(decimal):
    sexagesimal = []
    integer_part = int(decimal)
    decimal_part = decimal - integer_part

    while integer_part > 0:
        remainder = integer_part % 60
        sexagesimal.insert(0, remainder)
        integer_part //= 60

    for i in range(6):
        decimal_part *= 60
        sexagesimal.append(int(decimal_part))
        decimal_part -= int(decimal_part)

    return sexagesimal

2の平方根#

  • YBC 7289 - Wikipedia

%%ai_ask 
YBC 7289について

申し訳ありませんが、”YBC 7289”という用語については情報が不足しており、何についての情報が必要なのかを教えていただけますか？具体的な質問があればお答えできるかもしれません。

%%ai_continue
古代バビロニアの粘土版でイェール大学が保存しているもの

ありがとうございます。YBC 7289は、古代バビロニアの粘土板で、現在はイェール大学のバビロニア文書コレクションに所蔵されています。この粘土板には、数学的な問題やテーブルが記されており、古代バビロニアの数学や天文学の研究に貢献しています。特に、ピタゴラスの定理に関する最古の記録が含まれていることで知られています。

%%ai_continue
古代バビロニア人が2の平方根を求めた方法は

古代バビロニア人が2の平方根をどのように求めたかについては、正確な方法は分かっていませんが、いくつかの方法が提唱されています。

一つの方法は、ニュートン法に似た反復法を用いたものです。具体的には、以下の式を反復的に計算することで、2の平方根を求めました。

x_(n+1) = (x_n + 2/x_n) / 2

ここで、x_nはn回目の反復での近似値を表します。この方法は、古代バビロニアの数学者が用いた可能性があるとされています。

また、別の方法として、2の平方根を含む三角形の面積を求める方法があります。具体的には、底辺が1、斜辺が√2の直角三角形を考え、その面積を求めることで2の平方根を求めました。この方法は、古代エジプトでも用いられていたとされています。
